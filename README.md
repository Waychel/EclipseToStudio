# EclipseToStudio
Android Studio 使用Eclipse项目，仍保留项目结构，非导入转成gradle


目前Android项目基本都转成gradle，
但仍有不少停留在Eclipse或习惯于Eclipse生成的项目结构,
因某些因素项目只能保留还是Eclipse生成的源码结构，
但又想用Studio,
只能折腾一番。
